Zimbra Collaboration 10 Phishing ESET Antivirus

Alertas de Ransomware y APT

Ransomware Windows Server 2012 Windows Server 2016 EvilCorp 2.0

Nueva Variante de Ransomware "EvilCorp 2.0" Dirigida a Servidores Windows

Publicado: 2026-03-24

  • Identificada una nueva y agresiva variante del ransomware "EvilCorp", apodada 2.0, que ataca específicamente Windows Server 2012 y 2016.
  • La propagación se realiza a través de RDP comprometido, explotación de EternalBlue (en sistemas no parcheados) y correos de phishing con archivos adjuntos maliciosos.
  • Capaz de cifrar grandes volúmenes de datos rápidamente y evadir algunas soluciones EDR/AV.

Acción Recomendada: Implementar Multi-Factor Authentication (MFA) para todas las conexiones RDP. Reforzar la segmentación de red. Asegurar copias de seguridad inmutables y realizar simulacros de recuperación. Auditoría de sistemas Windows Server.

Recomendaciones Generales de Ciberseguridad

Gestión de Parches Seguridad Proactiva Infraestructura Crítica

Fortalecer la Gestión y Aplicación de Parches de Seguridad

Publicado: 2026-02-26

  • La aplicación oportuna de parches sigue siendo una de las defensas más efectivas contra la mayoría de las amenazas.
  • Esto incluye Fortinet (FortiOS, FortiGate, FortiAP, FortiMail), Proxmox VE, Dell (iDRAC, firmware), Windows, macOS, Ubuntu, AlmaLinux y RHEL.
  • Priorizar la aplicación de parches críticos en un plazo de 24-72 horas.

Acción Recomendada: Establecer un programa de gestión de parches robusto con procesos claros para la identificación, prueba y despliegue. Automatizar donde sea posible y verificar la aplicación exitosa. No descuidar los sistemas "offline" o ICS.

CISA Patch Management Guidance | NIST SP 800-40 Rev. 4
Concientización Ingeniería Social Phishing

Programas de Concientización y Capacitación Continua de Usuarios

Publicado: 2026-03-03

  • A pesar de las tecnologías avanzadas, el error humano sigue siendo un vector de ataque principal, especialmente a través de phishing e ingeniería social.
  • Los usuarios de sistemas Windows 10/11, macOS, y correos Zimbra son constantemente objetivo.
  • Una fuerza laboral informada es la primera línea de defensa.

Acción Recomendada: Implementar programas de capacitación de ciberseguridad continuos y obligatorios para todos los empleados. Realizar simulacros de phishing periódicos para evaluar y mejorar la capacidad de respuesta. Fomentar una cultura de seguridad.

SANS Security Awareness Training | NCSC Phishing Guidance
