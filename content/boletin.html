Incidentes Recientes

Compromiso de Perímetros con FortiGate No Parcheado

  • Múltiples incidentes donde atacantes explotaron una vulnerabilidad crítica de RCE en FortiOS para obtener acceso inicial a redes corporativas.
  • La explotación ha llevado a la instalación de backdoors y movimientos laterales dentro de las organizaciones afectadas.
  • Productos Afectados: FortiGate con versiones vulnerables de FortiOS (especialmente 6.x y 7.0.x no parcheadas).

Severidad: CRÍTICO

Acción Recomendada: Revisar los logs de FortiGate en busca de actividad sospechosa. Aislar y parchear inmediatamente los dispositivos FortiGate vulnerables.

FortiGuard PSIRT Update CISA Alert on Fortinet Exploitation

Explotación de SQL Injection en SQL Server Express 2008

  • Varios sitios web y aplicaciones que utilizan SQL Server Express 2008 backend han sido comprometidos a través de ataques de inyección SQL.
  • Los atacantes han logrado exfiltrar bases de datos y en algunos casos, ejecutar comandos remotos.
  • Productos Afectados: SQL Server Express 2008 R2 (y versiones anteriores) con aplicaciones vulnerables.

Severidad: ALTO

Acción Recomendada: Realizar un inventario de todas las instancias de SQL Server Express 2005/2008. Auditar aplicaciones para vulnerabilidades de inyección SQL. Considerar migración urgente a versiones más recientes.

Microsoft SQL Server Lifecycle OWASP SQL Injection Guide

Panorama Ransomware

Campañas de Ransomware Dirigidas a Windows Server 2012/2016

  • Grupos de ransomware han intensificado sus ataques contra infraestructuras que ejecutan Windows Server 2012 y 2016, aprovechando vulnerabilidades no parcheadas o configuraciones débiles.
  • Se observa un enfoque en la exfiltración de datos antes del cifrado, aumentando la presión sobre las víctimas.
  • Impacto: Interrupción de servicios críticos, pérdida de datos y daños a la reputación.

Severidad: CRÍTICO

Acción Recomendada: Asegurar que todos los servidores Windows Server 2012/2016 estén completamente actualizados y monitoreados. Implementar MFA en todos los accesos. Fortalecer las copias de seguridad.

CISA Advisory - Ransomware Activity Microsoft Security Blog

Amenazas de Ransomware en Entornos Legacy (Windows XP/2003)

  • A pesar de ser sistemas fuera de soporte, continúan siendo objetivos de variantes de ransomware diseñadas para explotar sus debilidades conocidas.
  • La ausencia de parches de seguridad para estos sistemas los convierte en puntos de entrada fáciles para los atacantes.
  • Impacto: Compromiso total del sistema, cifrado de datos y propagación a la red.

Severidad: ALTO

Acción Recomendada: Migrar urgentemente de sistemas Windows XP, Server 2003 y 2008. Aislar estos sistemas en segmentos de red si la migración no es posible inmediatamente. Mantener ESET Antivirus actualizado en estos equipos si están activos.

NCSC Guidance on EOL Software ESET - Windows XP EOL

Recomendaciones Generales

  • Gestión de Parches: Priorizar la aplicación de parches de seguridad críticos, especialmente para FortiGate, FortiMail, Proxmox, Zabbix, y todos los sistemas operativos (Windows Server, RHEL, AlmaLinux, Ubuntu, macOS).
  • Seguridad de Dispositivos de Borde: Asegurar que los dispositivos de perímetro como FortiGate estén siempre con las últimas actualizaciones y configuraciones robustas. Restringir el acceso a interfaces de administración como iDRAC y cámaras Dahua, manteniéndolas en redes aisladas y sin exposición a internet.
  • Monitorización Activa: Implementar y mantener una monitorización robusta con Zabbix para detectar anomalías y actividades sospechosas, especialmente en servidores Windows y sistemas Linux, así como en dispositivos de red.
  • Actualización de Sistemas Legacy: Realizar un plan urgente para la migración de sistemas operativos y bases de datos obsoletos como Windows XP, Server 2003/2008 y SQL Server Express 2005/2008. Si la migración no es posible, aplicar medidas de aislamiento estrictas y segmentación de red.
  • Defensa contra Ransomware: Mantener copias de seguridad robustas, inmutables y probadas. Implementar autenticación multifactor (MFA) en todos los servicios críticos y sensibilizar a los usuarios sobre ataques de ingeniería social y phishing. Asegurarse de que el antivirus ESET corporativo esté siempre actualizado y configurado correctamente en todos los endpoints.
  • Evaluación Continua: Realizar auditorías de seguridad periódicas y pruebas de penetración para identificar y corregir vulnerabilidades antes de que sean explotadas.
