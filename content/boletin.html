Leer más

Durante el mes de marzo de 2026, se han identificado vulnerabilidades significativas que requieren atención inmediata. La detección de una vulnerabilidad de ejecución remota de código (RCE) en Dell iDRAC (CVE-2026-26945) y múltiples fallos críticos en Zabbix que permiten inyección de comandos y SQL (WID-SEC-2026-0857) subraya la importancia de mantener actualizados los sistemas de gestión y monitoreo expuestos a la red.

El panorama de amenazas se mantiene activo en el frente del phishing, con campañas sofisticadas dirigidas a usuarios de Office 365 que explotan mecanismos de autenticación y la confianza en la infraestructura de Microsoft para el robo de credenciales y bypass de MFA. En cuanto a dispositivos IoT, las cámaras Dahua siguen siendo un vector de ataque común si no se gestionan adecuadamente las actualizaciones de firmware y las contraseñas. Aunque no se han reportado nuevas campañas de ransomware a gran escala específicas para nuestro stack en este período, el riesgo latente permanece, especialmente para los sistemas Windows y Windows Server fuera de soporte.

Incidentes Reportados y Actividad Observada

Vulnerabilidad de Ejecución Remota de Código en Dell iDRAC (CVE-2026-26945)

Internet-exposed RCE Observado Directa

Publicado: 2026-03-20

  • Una vulnerabilidad de control de procesos (CWE-114) en Dell Integrated Dell Remote Access Controller (iDRAC) 9 y 10 permite a atacantes con privilegios altos y acceso a la red adyacente ejecutar código arbitrario.
  • Afecta a iDRAC9 (14G) versiones anteriores a 7.00.00.181, iDRAC9 (15G y 16G) versiones anteriores a 7.20.10.50, e iDRAC10 (17G) versiones anteriores a 1.20.25.00.
  • iDRAC es una interfaz crítica para la administración remota de servidores, lo que hace que esta vulnerabilidad sea de gran preocupación para entornos de centros de datos empresariales.
Acción Recomendada: Revisar y actualizar inmediatamente el firmware de Dell iDRAC a las versiones parcheadas para mitigar esta vulnerabilidad. Implementar segmentación de red para restringir el acceso a las interfaces de gestión de iDRAC.
SentinelOne: CVE-2026-26945: Dell iDRAC RCE Vulnerability

Múltiples Vulnerabilidades Críticas en Zabbix (WID-SEC-2026-0857)

Internet-exposed RCE SQL Injection Info Leak Observado Directa

Publicado: 2026-03-24

  • CERT-Bund ha emitido una alerta sobre múltiples vulnerabilidades en Zabbix (CVSS 8.8) que permiten a atacantes remotos inyectar comandos de shell, realizar inyección SQL, divulgar información y causar denegación de servicio.
  • Afecta a varias versiones de Zabbix, incluyendo aquellas anteriores a 7.4.3, 7.0.19, 7.2.13, y 6.0.41.
  • La explotación exitosa puede llevar a un compromiso completo del sistema de monitoreo.
Acción Recomendada: Actualizar Zabbix a las últimas versiones parcheadas (7.4.3, 7.0.19, 7.2.13, 6.0.41 o posteriores según la versión utilizada). Implementar segmentación de red y controles de acceso estrictos para las instancias de Zabbix.

Campañas de Phishing que Evaden MFA en Office 365

Internet-exposed Phishing Explotado activamente Directa

Publicado: 2026-02-19 (última actividad)

  • Nuevas campañas de phishing abusan del registro de dispositivos OAuth para bypass la autenticación multifactor (MFA) en Microsoft 365, permitiendo acceso persistente a las cuentas de los usuarios.
  • Los atacantes engañan a los empleados para que registren un dispositivo controlado por el atacante en su cuenta, obteniendo tokens OAuth.
  • También se observan campañas que explotan la infraestructura legítima de Microsoft 365 y las malas configuraciones de inquilinos para enviar correos de phishing que parecen auténticos.
Acción Recomendada: Reforzar la capacitación en concienciación sobre ciberseguridad para los usuarios de Office 365. Implementar políticas de acceso condicional para limitar los tipos de dispositivos que pueden registrarse y monitorear activamente los cambios administrativos en el tenant de Microsoft 365. Restringir las aplicaciones a las que los usuarios pueden conectar su cuenta.

Nuevos CVEs y Actualizaciones de Seguridad

Bypass de Autenticación en FortiOS con FortiCloud SSO (CVE-2026-24858)

Internet-exposed Auth Bypass Explotado activamente Directa

Publicado: 2026-01-27

  • Una vulnerabilidad crítica de bypass de autenticación (CVSS 9.4) en FortiOS permite a un atacante no autenticado, con una cuenta FortiCloud y un dispositivo registrado, omitir la autenticación FortiCloud SSO.
  • Esto podría conducir a un compromiso total del sistema si la función FortiCloud SSO está habilitada.
  • Afecta a FortiOS 7.2.0 a través de 7.2.12.
Acción Recomendada: Actualizar FortiOS a la versión 7.2.13 o superior. Si FortiCloud SSO está habilitado, verificar las configuraciones de seguridad y considerar deshabilitar "Allow administrative login using FortiCloud SSO" si no es estrictamente necesario.

Vulnerabilidad de XSS Almacenado en Proxmox VE 8.4 (CVE-2025-57539)

Internal XSS Observado Directa

Publicado: 2025-09-09

  • Una vulnerabilidad de Cross-Site Scripting (XSS) almacenado en el campo "U2F Origin" de la configuración de Datacenter en Proxmox Virtual Environment (PVE) 8.4 permite a usuarios autenticados almacenar entrada maliciosa.
  • La carga útil se renderiza de forma insegura en la interfaz de usuario web y se ejecuta cuando otros usuarios la visualizan, lo que podría conducir al secuestro de sesiones.
Acción Recomendada: Aplicar las actualizaciones de seguridad de Proxmox VE tan pronto como estén disponibles para PVE 8.4 que aborden esta vulnerabilidad. Capacitar a los administradores sobre el riesgo de XSS y buenas prácticas al introducir datos en la interfaz web.

Elevación de Privilegios en Dell iDRAC Service Module (DSA-2026-077)

Internal Privilege Escalation Observado Directa

Publicado: 2026-02-10

  • El módulo de servicio Dell iDRAC (iSM) contiene una vulnerabilidad de control de acceso inapropiado que podría permitir a un atacante con bajos privilegios y acceso local escalar privilegios.
  • Afecta a Dell iDRAC Service Module (iSM) para Windows, versiones anteriores a 6.0.3.1, y para Linux, versiones anteriores a 5.4.1.1.
Acción Recomendada: Actualizar Dell iDRAC Service Module a las versiones parcheadas (6.0.3.1 para Windows, 5.4.1.1 para Linux o posteriores). Asegurar el acceso físico y lógico a los sistemas donde se ejecuta iSM.

Panorama de Ransomware

Sin reportes verificados

Actualizado: 2026-03-31

No se han detectado nuevos eventos de ransomware confirmados o campañas de alto impacto dirigidas específicamente a nuestro stack tecnológico definido durante el mes de marzo de 2026.

Sin embargo, la amenaza de ransomware sigue siendo persistente, especialmente para sistemas Windows y Windows Server fuera de soporte (XP, 7, 2003, 2008) que carecen de parches de seguridad y protecciones modernas. Se recomienda mantener una postura de seguridad proactiva.

Recomendaciones Operativas Clave

Actualización y Hardening de Sistemas

  • Fortinet: Asegurarse de que FortiOS esté actualizado a la versión 7.2.13 o posterior para mitigar la vulnerabilidad de bypass de autenticación CVE-2026-24858.
  • Dell iDRAC y iSM: Priorizar la actualización de firmware para iDRAC a las versiones parcheadas para CVE-2026-26945 y iDRAC Service Module a las versiones 6.0.3.1 (Windows) o 5.4.1.1 (Linux) o posteriores.
  • Proxmox VE: Aplicar los parches de seguridad para Proxmox VE 8.4 que aborden las vulnerabilidades XSS (CVE-2025-57539).
  • Windows y Windows Server: Mantener actualizados todos los sistemas operativos soportados (Windows 10, 11, Server 2012 R2, 2016) con los últimos parches de seguridad de Microsoft. Planificar la migración de sistemas operativos EOL (XP, 7, Server 2003, 2008, SQL Express 2005, 2008) a versiones soportadas.
  • Ubuntu Server 24.04 LTS: Implementar hardening siguiendo las guías CIS Benchmarks y asegurar la configuración de firewall (UFW), SSH (deshabilitar login root por contraseña), y Fail2Ban.
  • Red Hat Enterprise Linux 7 y 8: Monitorear y aplicar regularmente las Red Hat Security Advisories (RHSA) para mantener los sistemas parcheados.
  • Zabbix: Actualizar Zabbix a las últimas versiones parcheadas (7.4.3, 7.0.19, 7.2.13, 6.0.41 o posteriores).

Protección de Identidad y Dispositivos

  • Office 365: Reforzar la concienciación de usuarios sobre campañas de phishing que intentan evadir MFA. Implementar políticas de acceso condicional y monitorear el registro de dispositivos OAuth sospechosos.
  • ESET Antivirus: Asegurar la configuración óptima del antivirus corporativo con niveles de detección agresivos y considerar el uso de ESET Inspect para una visibilidad y respuesta mejoradas.
  • Cámaras Dahua: Mantener el firmware de todas las cámaras y HVRs Dahua actualizado. Forzar políticas de contraseñas robustas y únicas, y deshabilitar UPNP si no es esencial.
  • macOS: Asegurarse de que todos los dispositivos macOS (2015 en adelante) estén ejecutando la última versión del sistema operativo y que las actualizaciones de seguridad estén habilitadas.
