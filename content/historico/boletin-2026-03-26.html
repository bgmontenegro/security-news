Proxmox VE Privilege Escalation Actualización

Fallo de Escalada de Privilegios en Proxmox VE

Publicado: 2026-03-15

  • Se identificó una vulnerabilidad de escalada de privilegios local en Proxmox VE que permite a un usuario autenticado obtener acceso de root.

Windows Windows 10 Windows 11 SMB RCE

Vulnerabilidad Crítica en el Servicio SMB de Windows

Publicado: 2026-03-12

  • Microsoft ha lanzado un parche para una vulnerabilidad crítica de ejecución remota de código en el servicio Server Message Block (SMB).
  • La falla permite a un atacante no autenticado ejecutar código arbitrario en sistemas Windows 10 y Windows 11 vulnerables.
  • Se recomienda la aplicación urgente del parche debido al alto potencial de explotación remota.

Acción Recomendada: Aplique todas las actualizaciones de seguridad de Microsoft del "Patch Tuesday" de marzo de 2026 a sus sistemas Windows 10 y Windows 11.

Múltiples Fallas de Seguridad en Cámaras IP Dahua

Publicado: 2026-03-05

Dell iDRAC iDRAC13 Firmware

Múltiples CVEs Afectando Dell iDRAC hasta Generación 13

Publicado: 2026-03-10

  • Dell ha lanzado parches para corregir múltiples vulnerabilidades en iDRAC, incluyendo acceso no autenticado y escalada de privilegios, afectando hasta la generación 13.

Resurgimiento de una Variación de Ransomware Conti

Recomendaciones Generales

EOL Windows XP Windows Server 2003 Migración

Gestión y Migración Urgente de Sistemas al Final de su Vida Útil (EOL)

Publicado: 2026-03-01

  • Sistemas operativos como Windows XP, Windows 7, Windows Server 2003 y 2008, y SQL Server Express 2005 y 2008 son objetivos primarios para atacantes debido a la falta de parches de seguridad.
  • Mantener estos sistemas representa un riesgo inaceptable para la seguridad de la red y los datos corporativos.
  • La exposición continua aumenta drásticamente el riesgo de un incidente de seguridad mayor.

Acción Recomendada: Priorice la migración de todos los sistemas EOL a plataformas soportadas y actualizadas. Si la migración no es inmediatamente posible, aísle estos sistemas en segmentos de red ultra-restringidos, sin acceso a Internet, y con un monitoreo de seguridad intensivo.

MFA 2FA Seguridad Acceso

Implementación Universal de Autenticación Multifactor (MFA)

Publicado: 2026-02-27

  • La autenticación multifactor (MFA) sigue siendo una de las medidas más efectivas para prevenir el acceso no autorizado, incluso si las credenciales son robadas.
  • Muchos de los recientes incidentes de ciberseguridad podrían haberse mitigado o prevenido con una MFA robusta.
  • Es aplicable a una amplia gama de sistemas, incluyendo Fortinet, Zimbra, Zabbix, y acceso a sistemas operativos.

Acción Recomendada: Implemente MFA para todos los sistemas de acceso remoto, cuentas de administración, y siempre que sea posible para el acceso de usuarios a aplicaciones críticas como FortiMail, Zimbra, y Zabbix. Asegure que la MFA esté configurada correctamente y sea a prueba de phishing.
