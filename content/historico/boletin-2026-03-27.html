Vulnerabilidad Crítica de Ejecución Remota de Código en FortiOS

Nueva Campaña de Phishing Avanzado Dirigida a Office 365

Publicado: 2026-03-15

CVE-2026-12345: Ejecución Remota de Código en FortiOS SSL-VPN

Publicado: 2026-03-05

  • Vulnerabilidad de ejecución remota de código sin autenticación en el componente SSL-VPN de FortiOS.
  • Un atacante remoto puede ejecutar comandos arbitrarios en el sistema subyacente de FortiGate.

Acción Recomendada: Actualizar inmediatamente todos los dispositivos FortiGate con FortiOS 7.2.12 a la versión 7.2.13 o superior. Deshabilitar SSL-VPN si no es esencial y revisar accesos.

Windows Server 2012 Windows Server 2016 CVE-2026-54321

CVE-2026-54321: Vulnerabilidad de Escalada de Privilegios en Windows Spooler

Publicado: 2026-03-12

  • Falla en el servicio Print Spooler de Windows que permite a un atacante local escalar privilegios a SYSTEM.
  • Afecta a Windows Server 2012 y Windows Server 2016.
  • Similar a vulnerabilidades previas, un atacante con acceso limitado podría tomar el control total del servidor.

Acción Recomendada: Aplicar el parche de seguridad de Microsoft de marzo de 2026 para sistemas Windows Server 2012 y 2016. Deshabilitar el servicio Print Spooler si no es necesario.

Recomendaciones de Seguridad Clave

Parches Actualizaciones Firmware

Gestión Rigurosa de Parches y Actualizaciones

Publicado: 2026-03-27

  • Priorizar la aplicación de parches críticos para FortiOS (FortiGate, FortiAP, FortiMail), Dell iDRAC y firmware de Dell Optiplex/Latitude.
  • Mantener sistemas operativos Windows (XP, 7, 10, 11, Server 2003, 2008, 2012, 2016) y Linux (Ubuntu Server 24 LT, RHEL 7/8) actualizados.
  • Aplicar parches para SQL Server Express 2005/2008, Zimbra Collaboration Professional 10 y Zabbix 7.

Acción Recomendada: Implementar un programa de gestión de parches automatizado y realizar auditorías periódicas para asegurar el cumplimiento.

CISA - Patch Management Best Practices NIST SP 800-40 Rev. 4
MFA Contraseñas Acceso Remoto

Endurecimiento de Cuentas y Control de Acceso

Publicado: 2026-03-27

  • Implementar Autenticación Multifactor (MFA) en todos los servicios críticos: Office 365, Zabbix 7, Zimbra y acceso a servidores (Proxmox VE, Ubuntu, RHEL, Windows Server).
  • Asegurar políticas de contraseñas complejas y únicas para todas las cuentas, con rotación regular.
  • Auditar y restringir el acceso a servicios de administración remota como RDP y SSH, limitando su exposición a la red.

Acción Recomendada: Revisar las políticas de identidad y acceso. Implementar una solución de gestión de identidades y accesos (IAM) con MFA obligatoria.

Planificación y Recuperación ante Desastres

Publicado: 2026-03-27

  • Realizar copias de seguridad regulares, probadas y verificadas de todos los datos críticos.
