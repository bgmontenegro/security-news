Múltiples Vulnerabilidades Críticas en Windows Server y Cliente

Publicado: 2026-03-12

  • Una vulnerabilidad de ejecución remota de código (RCE) en el Servicio de Actualización de Windows Server (WSUS) afecta a Windows Server 2012 y versiones posteriores, y se encuentra bajo explotación activa.
  • Vulnerabilidades RCE en la herramienta de administración del Servicio de Enrutamiento y Acceso Remoto (RRAS) de Windows 11 han requerido un hotpatch de emergencia.

La explotación de estas fallas puede permitir a los atacantes obtener control total sobre los sistemas afectados.

Ransomware

Tendencias de Ransomware en Febrero y Marzo de 2026

Publicado: 2026-03-24

  • La actividad de ransomware se mantuvo elevada en febrero de 2026, con 694 incidentes reportados globalmente y 82 públicamente divulgados.
  • Los sectores más atacados incluyen servicios profesionales, manufactura, tecnologías de la información y salud.
  • Grupos como Qilin, Akira, Clop, INC Ransom, Play, DragonForce y Sinobi fueron los más activos en EE. UU. durante enero y febrero de 2026.
  • Las tendencias clave incluyen la multi-extorsión, el Ransomware as a Service (RaaS), y el uso de la IA para potenciar el phishing.

Acción recomendada: Implementar defensas multicapa, fortalecer la detección temprana, y capacitar al personal en reconocimiento de amenazas.

Vulnerabilidades y Exposiciones Comunes (CVEs)

Microsoft Windows Server

CVE-2025-59287: RCE en Windows Server Update Service (WSUS)

Publicado: 2025-10-14

  • Vulnerabilidad de deserialización insegura crítica en WSUS que permite la ejecución remota de código arbitrario.
  • Afecta a todas las versiones compatibles de Windows Server desde 2012 hasta 2025.
  • Se está explotando activamente y ha sido añadida al catálogo de vulnerabilidades conocidas explotadas (KEV) de CISA.

Acción recomendada: Priorizar el parcheo inmediato de los servidores WSUS. Requiere ninguna autenticación para su explotación remota.

Microsoft Windows 11

CVE-2026-25172, CVE-2026-25173, CVE-2026-26111: RCE en RRAS

Publicado: 2026-03-10

  • Múltiples vulnerabilidades de ejecución remota de código en la herramienta de administración del Servicio de Enrutamiento y Acceso Remoto (RRAS) de Windows.
  • La explotación podría permitir a un atacante ejecutar código arbitrario o interrumpir las operaciones del sistema.
  • Se emitió un hotpatch fuera de banda (KB5084597) en marzo de 2026 para versiones empresariales de Windows 11.

Acción recomendada: Aplicar el hotpatch KB5084597 o las actualizaciones de seguridad de marzo de 2026. Se requiere interacción del usuario autenticado en el dominio para la explotación.
