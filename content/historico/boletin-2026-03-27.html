Actualización Crítica

Múltiples Vulnerabilidades Críticas en Productos Microsoft

Publicado: 2026-03-10

  • Microsoft lanzó el Patch Tuesday de marzo de 2026, abordando 84 vulnerabilidades, incluyendo ocho críticas.
  • Se incluyen fallas de ejecución remota de código en Microsoft Office (CVE-2026-26110, CVE-2026-26113) y divulgación de información en Microsoft Excel (CVE-2026-26144).
  • Una vulnerabilidad de ejecución remota de código en Microsoft SharePoint (CVE-2026-20963) ha sido confirmada como activamente explotada y añadida al catálogo KEV de CISA.

Acción Recomendada: Aplicar todas las actualizaciones de seguridad de Microsoft de marzo de 2026 de inmediato. Priorizar la actualización de sistemas Windows, Office y SharePoint.

Actualización de Seguridad

Vulnerabilidades en Dell iDRAC

Publicado: 2026-03-20

  • Se han identificado múltiples vulnerabilidades en Dell iDRAC9 y iDRAC10, incluyendo divulgación de información (CVE-2026-26948) y ejecución remota de código (CVE-2026-26945).
  • Una vulnerabilidad de Path Traversal (CVE-2020-5366) en Dell iDRAC9, aunque antigua, sigue siendo un tema de discusión.

Acción Recomendada: Actualizar el firmware de iDRAC a las versiones parcheadas según el aviso de seguridad de Dell DSA-2026-113 y DSA-2020-128. Implementar controles de acceso de red estrictos para las interfaces de administración de iDRAC.

Incidentes Reportados

Incidente

CISA Alerta sobre Ciberataque a Sistemas de Gestión de Endpoints

Publicado: 2026-03-18

  • CISA ha emitido una alerta sobre actividad cibernética maliciosa dirigida a sistemas de gestión de endpoints de organizaciones estadounidenses, incluyendo un ataque a una empresa de tecnología médica que afectó su entorno de Microsoft.
  • El incidente subraya la importancia de fortalecer las configuraciones de los sistemas de gestión de endpoints para defenderse contra el uso indebido de software legítimo.

Acción Recomendada: Revisar y aplicar las mejores prácticas de seguridad para Microsoft Intune y otros software de gestión de endpoints. Implementar el principio de menor privilegio, MFA resistente a phishing y controles de acceso privilegiado.

CISA Alert: Endpoint Management System Hardening

Vulnerabilidades (CVEs)

Microsoft Office

CVE-2026-26110 y CVE-2026-26113: Ejecución Remota de Código Crítica

Publicado: 2026-03-10

  • Dos vulnerabilidades críticas de ejecución remota de código en Microsoft Office que podrían permitir a un atacante no autorizado ejecutar código localmente.
  • La vulnerabilidad CVE-2026-26110 es un problema de confusión de tipos, y CVE-2026-26113 es una desreferencia de puntero no confiable.

Acción Recomendada: Aplicar las actualizaciones de seguridad de Microsoft de marzo de 2026 para Office y sistemas Windows afectados.

Microsoft Security Advisory CVE-2026-26110 Microsoft Security Advisory CVE-2026-26113
Microsoft Excel

CVE-2026-26144: Vulnerabilidad de Divulgación de Información Crítica

Publicado: 2026-03-10

  • Vulnerabilidad crítica de divulgación de información que afecta a Microsoft Excel, debido a una neutralización incorrecta de la entrada.
  • Un atacante no autorizado podría divulgar información confidencial en los sistemas afectados.

Acción Recomendada: Instalar las actualizaciones de seguridad de Microsoft de marzo de 2026 para Excel.

Microsoft Security Advisory CVE-2026-26144
Microsoft .NET

CVE-2026-26127: Vulnerabilidad de Denegación de Servicio (Zero-Day)

Publicado: 2026-03-10

  • Vulnerabilidad de denegación de servicio en .NET (CVE-2026-26127), que podría permitir a un atacante no autorizado interrumpir el servicio.
  • Esta vulnerabilidad fue divulgada públicamente antes de la publicación del parche, aunque no hay evidencia de explotación activa.

Acción Recomendada: Aplicar las actualizaciones de seguridad de Microsoft de marzo de 2026 para los componentes de .NET.

Microsoft Security Advisory CVE-2026-26127
Microsoft SharePoint / Office 365

CVE-2026-20963: Ejecución Remota de Código Activamente Explotada

Publicado: 2026-03-24

  • Vulnerabilidad de ejecución remota de código en Microsoft SharePoint debido al manejo inadecuado de entradas serializadas, con explotación activa confirmada.
  • Esta vulnerabilidad fue añadida al catálogo de Vulnerabilidades Explotadas Conocidas (KEV) de CISA el 18 de marzo de 2026.

Acción Recomendada: Parchear las instancias de Microsoft SharePoint inmediatamente, especialmente aquellas accesibles externamente. Considerar que cualquier instancia expuesta o sin parchear está siendo atacada activamente.

Microsoft Security Advisory CVE-2026-20963 SentinelOne - SharePoint Deserialization RCE
Dell iDRAC

CVE-2026-26948: Divulgación de Información en iDRAC9

Publicado: 2026-03-20

  • Vulnerabilidad de divulgación de información en Dell Integrated Dell Remote Access Controller 9 (iDRAC9) debido a información de depuración no eliminada.
  • Un atacante con altos privilegios y acceso remoto podría explotar esta vulnerabilidad para obtener acceso a datos confidenciales del sistema.

Acción Recomendada: Actualizar el firmware de iDRAC a las versiones parcheadas según el aviso de seguridad de Dell DSA-2026-113.

Dell Security Advisory DSA-2026-113
Dell iDRAC

CVE-2026-26945: Ejecución Remota de Código en iDRAC

Publicado: 2026-03-20

  • Vulnerabilidad de Control de Procesos (CWE-114) en Dell Integrated Dell Remote Access Controller (iDRAC) que afecta a múltiples generaciones de servidores.
  • Un atacante con altos privilegios y acceso a la red adyacente podría ejecutar código arbitrario en sistemas iDRAC vulnerables.

Acción Recomendada: Actualizar el firmware de iDRAC a las versiones parcheadas especificadas en el aviso de seguridad de Dell DSA-2026-113. Asegurar el aislamiento de las interfaces de gestión de iDRAC en VLANs dedicadas.

Dell Security Advisory DSA-2026-113
Dell iDRAC9

CVE-2020-5366: Path Traversal en iDRAC9

Publicado: 2026-03-04 (Discusión actualizada)

  • Vulnerabilidad de Path Traversal en el firmware de Dell EMC iDRAC9 (versiones anteriores a 4.20.20.20), que permite a un usuario autenticado con bajos privilegios acceder a archivos arbitrarios.
  • Aunque es una vulnerabilidad de 2020, se sigue discutiendo su impacto en entornos empresariales.

Acción Recomendada: Actualizar el firmware de iDRAC9 a la versión 4.20.20.20 o posterior. Implementar controles de acceso a la red estrictos para limitar el acceso a la interfaz de gestión de iDRAC9.

Dell Security Advisory DSA-2020-128

Recomendaciones Generales

Priorizar Actualizaciones de Seguridad

Actualizado: 2026-03-27

  • Mantener todos los sistemas operativos, firmware (especialmente iDRAC de Dell) y aplicaciones (Microsoft Office, SharePoint) actualizados con los últimos parches de seguridad.
  • Prestar especial atención a los boletines de seguridad mensuales de los fabricantes y aplicar las actualizaciones críticas de inmediato.

Acción Sugerida: Implementar un proceso de gestión de parches robusto y automatizado para minimizar la ventana de vulnerabilidad.

Fortalecer la Gestión de Endpoints y Office 365

Actualizado: 2026-03-27

  • Configurar de forma segura los sistemas de gestión de endpoints y entornos de Microsoft, siguiendo las mejores prácticas de CISA y Microsoft.
  • Asegurar el uso del principio de menor privilegio, implementar MFA resistente a phishing y controles de acceso privilegiado.

Acción Sugerida: Realizar auditorías de configuración de seguridad en sistemas de gestión de endpoints y plataformas de colaboración para identificar y remediar posibles debilidades.

Aislamiento de Interfaces de Gestión

Actualizado: 2026-03-27

  • Aislar las interfaces de gestión como iDRAC en redes dedicadas (VLANs) con reglas de firewall estrictas, limitando el acceso solo a administradores autorizados.
  • Minimizar la exposición de estas interfaces a redes no confiables o a internet.

Acción Sugerida: Revisar y ajustar las configuraciones de red y firewall para garantizar el aislamiento adecuado de los componentes críticos de la infraestructura.
