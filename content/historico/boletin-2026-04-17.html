Leer más

Durante este período, el panorama de amenazas sigue destacando la sofisticación de los ataques de ingeniería social, especialmente el phishing. La inteligencia artificial está siendo ampliamente utilizada por los actores de amenazas para crear campañas más convincentes y a gran escala, dirigiéndose a credenciales de servicios en la nube como Office 365 a través de múltiples canales (email, SMS, redes sociales). La actividad de ransomware, aunque no ha experimentado un pico reciente, se ha asentado en un nivel alto y constante, con un énfasis creciente en la extorsión de datos y ataques a la cadena de suministro. Se observa una explotación activa de vulnerabilidades en productos clave del stack, como Fortinet FortiClient EMS y Microsoft SharePoint Server, lo que subraya la necesidad de aplicar parches de seguridad de manera prioritaria. Además, se han lanzado actualizaciones de seguridad importantes para sistemas operativos como Windows y macOS, y para software como ESET Antivirus y Office 365, que abordan múltiples fallas de seguridad y mejoran las capacidades defensivas. El riesgo latente para sistemas operativos y bases de datos obsoletos como Windows XP/Server 2003/2008 y SQL Server Express 2005/2008 persiste, dado su historial de vulnerabilidades sin parchear y la falta de soporte oficial.

  • CISA ha añadido CVE-2026-21643 (Fortinet FortiClient EMS SQL Injection) y CVE-2026-32201 (Microsoft SharePoint Server Improper Input Validation) a su catálogo de vulnerabilidades explotadas conocidas (KEV).

  • CVE-2026-32201 podría permitir a un actor de amenazas ver o modificar información sensible.

Acción Recomendada: Aplicar inmediatamente los parches de seguridad disponibles para FortiClient EMS y Microsoft SharePoint Server. Priorizar la remediación de estas vulnerabilidades según las indicaciones de CISA.

Campaña de Phishing "Operation GhostMail" explotando Zimbra XSS

Correo / Phishing Info Leak Explotado activamente Directa

Publicado: 2026-03-19

  • CISA advirtió sobre la explotación de CVE-2025-66376, una vulnerabilidad XSS almacenada en la UI Clásica de Zimbra Collaboration Suite (ZCS).
  • Los atacantes utilizan directivas CSS @import en emails HTML para inyectar y ejecutar scripts maliciosos.
  • Esta campaña ("Operation GhostMail") se dirige a la recolección de credenciales, tokens de sesión, códigos de recuperación 2FA y contenido de buzones.
  • La vulnerabilidad fue corregida en ZCS 10.0.18 y 10.1.13 (Noviembre de 2025), pero la advertencia de CISA en Marzo de 2026 indica explotación continua de sistemas sin parchar.

Acción Recomendada: Asegurar que todas las instancias de Zimbra Collaboration Suite estén actualizadas a las versiones 10.0.18 o superior, o 10.1.16 o superior. Reforzar la concienciación de los usuarios sobre correos electrónicos sospechosos, incluso sin archivos adjuntos o enlaces obvios.

Nuevas Vulnerabilidades (CVEs)

CVE-2026-23921: Inyección SQL en Zabbix API

Internal Info Leak Observado Directa

Publicado: 2026-03-27

  • Vulnerabilidad de inyección SQL ciega en el componente API de Zabbix (include/classes/api/CApiService.php).
  • Un usuario con bajos privilegios y acceso a la API puede exfiltrar datos arbitrarios de la base de datos utilizando técnicas basadas en tiempo.
  • Esto podría conducir a la divulgación de identificadores de sesión y al compromiso de cuentas de administrador.
  • Afecta a Zabbix 7.0.x y versiones anteriores.

Acción Recomendada: Monitorear los avisos de seguridad de Zabbix (ZBX-27640) y aplicar el parche oficial tan pronto como esté disponible. Implementar estrictos controles de acceso para usuarios con bajos privilegios a la API de Zabbix.

CVE-2026-23919: Divulgación de Información en Zabbix Server/Proxy

Internal Info Leak Teórico Directa

Publicado: 2026-03-27

  • Vulnerabilidad de divulgación de información de alta gravedad en Zabbix Server y Proxy debido a la reutilización de contextos JavaScript (Duktape).
  • Permite a un administrador regular (no superadministrador) de Zabbix filtrar datos sensibles de hosts a los que no debería tener acceso.
  • La explotación requiere acceso adyacente a la red y privilegios de administrador en Zabbix.

Acción Recomendada: Monitorear los avisos de seguridad de Zabbix (ZBX-27638) y aplicar el parche oficial. Reforzar la configuración de privilegios y el principio de mínimo privilegio para los administradores de Zabbix.

CVE-2026-33372 y CVE-2026-33373: Vulnerabilidades CSRF en Zimbra Collaboration

Internet-exposed Auth Bypass Teórico Directa

Publicado: 2026-03-20 / 2026-03-30

  • CVE-2026-33372: Vulnerabilidad de Cross-Site Request Forgery (CSRF) en Zimbra Webmail debido a la validación incorrecta de tokens CSRF, aceptando tokens en el cuerpo de la solicitud.
  • CVE-2026-33373: Otra vulnerabilidad CSRF en Zimbra Web Client por la emisión de tokens de autenticación sin protección CSRF durante transiciones de estado de cuenta específicas.
  • Ambas afectan a Zimbra Collaboration (ZCS) 10.0 y 10.1.
  • Un atacante podría engañar a un usuario autenticado para que envíe una solicitud manipulada, realizando acciones no autorizadas.

Acción Recomendada: Revisar los avisos de seguridad de Zimbra para parches específicos para ZCS 10.0 y 10.1. Implementar medidas de seguridad adicionales para prevenir ataques CSRF, como el uso de cabeceras SameSite cookies y mecanismos de protección contra clickjacking.

Actividad de Ransomware

Persistencia de la Amenaza Ransomware y Nuevas Tácticas

Internet-exposed Data Leak Explotado activamente Indirecta

Publicado: 2026-04-16

  • La actividad de ransomware se ha estabilizado en un "nuevo nivel normal" elevado, sin las caídas estacionales observadas en años anteriores.
  • Estados Unidos sigue siendo el objetivo principal, con el 51% de las víctimas.
  • Grupos como "The Gentlemen" han visto un aumento significativo en su actividad, mientras que otros como Qilin han disminuido, mostrando una reconfiguración de los actores principales.
  • Las tendencias incluyen la multi-extorsión, el crecimiento de RaaS, y el enfoque en ataques a la cadena de suministro y a MSPs.
  • La IA es una herramienta clave para supercargar el phishing y la ingeniería social, facilitando los ataques de ransomware.

Acción Recomendada: Fortalecer las defensas contra el phishing y la ingeniería social, especialmente las capacitaciones sobre deepfakes y ataques multicanal. Implementar segmentación de red, copias de seguridad inmutables y planes de respuesta a incidentes robustos. Mantener los sistemas y software actualizados para mitigar vectores de acceso inicial comunes.
