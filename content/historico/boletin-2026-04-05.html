Radar Operativo Diario

  • Internet-exposed: Alta actividad de escaneo y explotación de servicios expuestos, especialmente Fortinet y Proxmox.
  • Correo / Phishing: Campañas continuas y sofisticadas dirigidas a credenciales de Office 365 y Zimbra.
  • IoT / Cámaras: Compromiso persistente de dispositivos Dahua con configuraciones débiles o firmware desactualizado.

Leer más

El panorama de amenazas se mantiene dinámico con un enfoque sostenido en la explotación de credenciales de acceso y vulnerabilidades en servicios expuestos a Internet. Observamos un incremento en los intentos de compromiso a firewalls FortiGate y plataformas de virtualización Proxmox VE, buscando puntos de entrada inicial para movimientos laterales. Las campañas de phishing continúan siendo un vector primario para la exfiltración de credenciales, impactando directamente a entornos de Office 365 y Zimbra. En el frente de IoT, los dispositivos Dahua desactualizados o mal configurados siguen siendo un blanco fácil para la conformación de botnets o accesos no autorizados. La amenaza de ransomware persiste, con grupos apuntando a infraestructura de servidor Windows y RHEL, incluyendo bases de datos SQL Server Express, por su valor y potencial interrupción de operaciones.

A pesar de no identificarse nuevos CVEs críticos con explotación activa *confirmada* específicamente para el período reciente, los riesgos latentes asociados a configuraciones por defecto, sistemas sin parches y credenciales débiles se mantienen como las principales vías de ataque. La extensión de soporte para Windows Server 2012 hasta octubre de 2026 y para Windows 10 hasta la misma fecha, a través de ESUs, es una señal importante de riesgos continuos en sistemas más antiguos que requieren atención especial.

Nuevas Vulnerabilidades (CVEs)

Actualizaciones de seguridad de Windows Server 2012 R2

Internal RCE Explotado activamente Directa

Publicado: 2026-02-10

  • Microsoft lanzó una actualización de seguridad crítica para Windows Server 2012 R2 en febrero de 2026, abordando múltiples vulnerabilidades con una calificación de riesgo crítico.
  • Una de las vulnerabilidades abordadas tiene un CVSSv3 base score de 10.0 y se considera activamente explotable.
  • Los certificados de Secure Boot utilizados por la mayoría de los dispositivos Windows caducarán a partir de junio de 2026, lo que podría afectar la capacidad de arranque seguro si no se actualizan a tiempo.

Acción Recomendada: Aplicar inmediatamente la actualización de seguridad KB5075970 para Windows Server 2012 R2 y KB5079233 (Servicing stack update) si aún no se ha hecho. Preparar los servidores para las actualizaciones de certificados de Secure Boot.

KB5075970: Windows Server 2012 R2 Security Update (February 2026) [Tenable] KB5079233: Servicing stack update for Windows Server 2012 R2: March 10, 2026 [Microsoft] Microsoft Security Update Guide

Recomendaciones Operativas Clave

Gestión de Parches y Actualizaciones Críticas

Internal Privilege Escalation Observado Directa

Publicado: 2026-04-05

  • Asegurar que FortiOS en FortiGate y FortiAP esté en la versión 7.2.12 o superior, monitoreando activamente los avisos de FortiGuard Labs.
  • Mantener Proxmox VE 8.4.16 o superior. Es crucial revisar la documentación de seguridad de Proxmox.
  • Aplicar todas las actualizaciones de seguridad de Microsoft para Windows 10/11 y Windows Server (2012, 2016). Windows 10 tiene ESU hasta Octubre de 2026, y Windows Server 2012 hasta Octubre de 2026.
  • Actualizar firmware de Dell Optiplex, Latitude e iDRAC (hasta Gen 13). Es fundamental suscribirse a los avisos de seguridad de Dell.
  • Para Red Hat Enterprise Linux 7 y 8, consultar las guías de seguridad y aplicar parches de software regularmente.

Acción Recomendada: Priorizar la revisión y aplicación de parches y actualizaciones de firmware. Establecer un ciclo de parcheo riguroso y automatizado para sistemas operativos y plataformas clave. Suscribirse a los avisos de seguridad de los proveedores.

Implementación y Refuerzo de MFA

Internet-exposed Auth Bypass Explotado activamente Directa

Publicado: 2026-04-05

  • Implementar y/o reforzar la autenticación multifactor (MFA) en Office 365, Zimbra Collaboration Professional 10+ y VPNs de FortiGate.
  • Priorizar métodos de MFA resistentes al phishing, como aplicaciones autenticadoras, sobre SMS.
  • Para FortiGate, la configuración de 2FA para el acceso de administrador y VPN es una medida de seguridad crítica.

Acción Recomendada: Auditar la implementación de MFA en todos los servicios y sistemas que lo soporten. Educar a los usuarios sobre los riesgos de phishing y la importancia de MFA.

Hardening de Dispositivos IoT y Cámaras Dahua

Internet-exposed RCE Observado Directa

Publicado: 2026-04-05

  • Cambiar contraseñas por defecto y utilizar credenciales robustas en cámaras y HVRs Dahua.
  • Segmentar redes para IoT y cámaras, limitando su exposición a Internet y evitando el acceso directo desde la WAN.
  • Mantener el firmware de las cámaras y HVRs Dahua actualizado.
  • Deshabilitar servicios innecesarios como UPNP, P2P, SNMP si no se utilizan.

Acción Recomendada: Realizar una auditoría de configuración de todos los dispositivos Dahua. Considerar implementar aislamiento de red y VPN para el acceso remoto.

Dahua Security Best Practices (Wiki) Dahua's Encrypted Network Cameras Explained [CCTV-MALL]

Monitoreo y Detección con Zabbix

Internal Info Leak Teórico Directa

Publicado: 2026-04-05

  • Asegurar que Zabbix 7 esté configurado para monitorear anomalías en el tráfico de red y el comportamiento de los servidores, incluyendo RHEL y Ubuntu Server 24.04 LTS.
  • Mantener la plataforma Zabbix actualizada. Se recomienda revisar las notas de actualización y realizar backups antes de cualquier upgrade.

Acción Recomendada: Verificar la configuración de alertas en Zabbix para incidentes de seguridad. Asegurar que los agentes de Zabbix estén actualizados y que la base de datos subyacente (MySQL o PostgreSQL) cumpla con los requisitos de la versión 7 de Zabbix.
